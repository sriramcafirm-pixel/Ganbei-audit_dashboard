▶ How to link a Google Sheet
Step 1 — Get Sheet ID: Open your Google Sheet. The ID is in the URL between /d/ and /edit.
docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit

Step 2 — Create API Key: Go to console.cloud.google.com → APIs & Services → Credentials → Create API Key. Enable the Sheets API and restrict the key to it.

Step 3 — Apps Script (for write): In your Sheet go to Extensions → Apps Script, paste your web app code, deploy as Web App (Anyone can access), and copy the URL.
